Digital Circuits

A demo/prototype for a circuit building puzzle game.

This is an early prototype for a circuit building puzzle game. The idea was to get experience working with UI elements as game mechanics, and to try and make interesting and fun puzzles out of real world problems. In the future, I'd like to expand the game to include more complex circuits using components you've already built, eventually working up to designs like a chronometer or basic memory.
