Butter Late than Never

A 2D platformer made in 9 days for the 14th Pirate Software Game Jam

As my first platformer, I was overall pleased with the outcome, given the time constraint. I found the general mechanics we're received well, but some elements needed to be explained more clearly and were only discovered after some trial and error. I found the level design to be more difficult than actually programming the game, and was mostly just experimenting with what ideas I could add that felt fun to me. In future projects, I'd like to set aside more time for designing levels and start building levels concurrently with designing the basic mechanics.
